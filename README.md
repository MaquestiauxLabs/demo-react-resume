- Node.js >= 18.0.0

### Installation

```bash
# Clone the repository
git clone https://github.com/MaquestiauxLabs/demo-react-resume.git
cd demo-react-resume

# Install dependencies
npm install

# Start development server
npm run dev
```

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run lint:fix` - Fix ESLint issues
- `npm run format` - Format code with Prettier

## ğŸ“ Customization

The resume content can be easily customized by modifying the component files in the `src/` directory. The project uses a modular structure where:
